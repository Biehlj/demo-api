## Getting Started

1. Clone the repository:

    ```bash
    git clone https://github.com/BiehlJ/demo-api.git
    ```

2. Navigate to the project directory:

    ```bash
    cd demo-api
    ```

3. Build the project:

    ```bash
    dotnet build
    ```

4. Run the project:

    ```bash
    dotnet run
    ```

5. Open your web browser and navigate to `https://localhost:5001` to access the API.
